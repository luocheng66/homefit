# 蜕变·HomeFit 🏋️

在家锻炼规划与追踪软件，让普通人无需专业基础即可科学健身。

## 功能特性

- 📋 **智能问卷引导** - 首次使用通过问卷收集身体数据，量身定制计划
- 💪 **科学训练计划** - 支持室内/室外两种模式，根据健康状况规避风险
- ⚖️ **体重追踪** - 每日双体重记录(起床/睡前)，智能趋势分析
- 📊 **数据可视化** - 体重曲线、趋势线、里程碑标记
- 🍃 **饮食记录** - 红黄绿灯饮食标签，简单直观
- 🔥 **连续打卡** - 徽章系统，激励坚持
- 📝 **每周周报** - 自动生成训练报告，支持导出
- 💡 **智能建议** - 根据体重变化给出科学建议

## 使用方法

直接在浏览器打开 `index.html` 即可使用。

## 技术栈

- 纯前端实现，无需后端
- HTML5 + CSS3 + JavaScript
- Canvas 图表绘制
- LocalStorage 数据持久化
